Перелік основних команд, якi потрібно записати:

  • git config --global user.name blabla
  • git config --global user.email blabal@bla
  • git init
  • git status
  • git add
  • git commit
  • git log
  • git checkout
  • git branch
  • git push
  • git pull
  • git merge
  • git clone

gitignor file - це текстовий файл, поміщений у ваш репозиторій git, який повідомляє git не відстежувати певні файли та папки,
які ви не хочете завантажувати у ваш головний репозиторій.

Записати такі команди:

  • git revert
  • git reset
  • git rebase
  • git cherry-pick
  • git stash
